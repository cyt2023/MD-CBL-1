# Executive Summary

This project evaluates three charging strategies for a shared EV fleet in Eindhoven: baseline, smart priority, and agentic LLM.

- Baseline cost: EUR 88.473
- Smart priority cost: EUR 93.784
- Agentic LLM cost: EUR 89.917
- Agentic unmet demand: 4
- Real LLM used: False
- Fallback heuristic used: True

The agentic scheduler is evaluated by quantitative operational metrics and validated decision traces, not by subjective language quality.
